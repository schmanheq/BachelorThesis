# Reconstructing epidemiological data using VGAE

> **Bachelor Thesis** - Freie Universität Berlin, Department of Computer Science and Mathematics
> **Author:** Tuan Anh Ta | **Year:** [March 2026]

## 📌 Introduction

Welcome to the repository for my bachelor thesis, titled *"Reconstructing epidemiological data using Variational Graph Autoencoder"*. 

This project explores how suited Variational Graph Autoencoder are for the reconstruction of epidemiological data following the SIR-Model dynamics. The primary goal of this research is to establish a VGAE model which performs matrix reconstruction on a feature matrix, which is based of the patients infection histories.

### 🎯 Research Questions
* **[Question 1]:** How well can Variational Graph Autoencoder reconstruct missing data from an epidemiological
dataset?
* **[Question 2]:** How well does it perform against other methods?
* **[Question 3]:** How does it perform when the amount of data or missingness changes?

This repository contains the source code used for the VGAE and data creation used to conduct the research and generate the final thesis results, as well as the bachelor thesis itsself.
